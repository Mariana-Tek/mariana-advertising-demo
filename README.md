# Mariana Advertising Demo
[![Deploy](https://www.herokucdn.com/deploy/button.svg)](https://heroku.com/deploy)

A simple example of how to use the Mariana Tek Advertising API.

## System Dependencies
- Node.js v10.15

## Installation
```bash
$ nvm install
$ npm install
```

## Local Development
```bash
$ npm run dev
```
